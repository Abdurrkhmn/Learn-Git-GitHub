1. Create new folder
In terminal: mkdir
2. Initializing a Git Repository 
In terminal: git init
3. Create random file
In terminal: touch index.js
3. Check file status
In terminal: git status
4. Add file to staging
In terminal: git add .
5. Committing the staging file
In terminal: git commit -m "Your Commit Message,
determines your changes"
6. Displaying Differences (melihat pembaruan)
In terminal: git diff filename/file.txt
7. Showing Git Commit Logs
In terminal: git log
